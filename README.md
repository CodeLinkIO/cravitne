# crawler service

crawl articles from popular news publishers in Vietnam:

- VnExpress
- TuoiTre
- NguoiLaoDong

## Tech stack

- Flask
- Scrappy

## Installation

Prerequisites:

- Python 3.8+
- Docker
- Docker Compose

### Run the service

```bash

docker-compose build

docker-compose up -d

```

### Run locally

- Run file `main_local.py`
